📖 Conceptos E-R

Entidad

Objeto o concepto del mundo real del que se guarda información. Se representa con un rectángulo. Ej: CLIENTE, PRODUCTO, PELÍCULA.

Atributo

Propiedad o característica de una entidad o relación. Se representa con un óvalo. Ej: nombre, fecha, precio. Los atributos clave se subrayan.

Relación

Vínculo o asociación entre entidades. Se representa con un rombo. Suele ser un verbo. Ej: compra, realiza, pertenece.

Cardinalidad

Indica cuántas instancias de una entidad se relacionan con otra. Se representa con "1" (uno) o "N" (muchos). Ej: 1:N significa "uno a muchos".

Atributo Compuesto

Atributo que puede dividirse en partes más pequeñas. Ej: Dirección (Calle, Número, Ciudad). Se anota como Nombre(Componente1, Componente2).

Atributo Multivaluado

Atributo que puede tener múltiples valores para la misma entidad. Se representa con doble borde. Ej: teléfonos, idiomas, actores de una película.

Atributo Derivado

Atributo cuyo valor se puede calcular a partir de otros. Se representa con borde punteado. Ej: edad (se calcula de la fecha de nacimiento).

Atributo de Relación

Atributo que describe información del vínculo entre entidades, no de las entidades mismas. Se conecta al rombo de relación. Ej: fecha_compra en "cliente compra producto".

📖 Glosario de E-R

Entidad

Objeto o concepto del mundo real del que se guarda información. Ej: CLIENTE, AUTO, PRODUCTO.

Relación

Vínculo o acción entre dos o más entidades. Suele ser un verbo. Ej: tiene, realiza, recibe.

Atributo simple

Dato que describe a una entidad o relación. No se puede dividir. Ej: nombre, fecha, precio.

Atributo clave (PK)

Identifica de forma única a cada instancia de una entidad. Se subraya. Ej: ci, id_cliente, isbn.

Atributo multivaluado

Puede tener varios valores para la misma entidad. Se dibuja con doble borde. Ej: teléfonos, actores.

Atributo compuesto

Se puede dividir en partes más pequeñas. Ej: Nombre_completo → Primer_nom + Primer_ape.

Atributo de relación

Describe información propia del vínculo entre entidades (no de las entidades en sí). Se conecta al rombo. Ej: fecha_entrada, observación en la relación "recibe".
